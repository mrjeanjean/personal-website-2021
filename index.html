Gaëtan Chesneau Développeur full-stack et formateur développeur web

Basé à Lille, je suis développeur front-end et back-end depuis plus de 15 ans. J'ai travaillé près de 6 ans dans divers agences de communication. Je travaille en freelance depuis 10 ans maintenant et la majorité du temps pour des agences sur divers types de projets.

J'ai travaillé comme formateur (développeur web JS/PHP) pour l'école numérique 3W Academy. Je suis disponible pour renouveler l'expérience au sein d'une école ou d'un organisme de formation à distance ou dans la métropole lilloise.

Intégrateur Développeur front-end senior css et javascript

HTML // CSS

Intégrateur HTML/CSS. Utilisateur forcené de SASS, je préfère mettre en place la méthodologie BEM pour le nommage des sélecteurs CSS. Je peux me greffer sur un projet Boostrap ou Foundation, mais je leur préfère du code fait main sans framework pour débuter les projets.

Javascript

Développeur Javascript, je code en Javascript natif (ou vanilla). Webpack, ViteJS, Rollup m'aident à coder le javascript dans sa version ES2020 et/ou en Typescript, tout en restant accessible aux navigateurs les plus anciens.

Vue JS

J'utilise Vue JS pour ajouter du contenu dynamique au sein d'une page web quand le javascript vanilla montre à mon sens ses limites. Pour les applications web plus ambitieuses, je me tourne le plus souvent vers React JS.

React JS

React JS, Redux me sont utiles pour développer des environnements dynamiques et complexes : routing, échanges avec API, interactions utilisateurs. Je suis également familier avec React Native, bien que pas réellement un expert.
